## 10.4 风险测度

	除了估值之外，风险管理是随机方法和模拟的另一个重要应用领域。
	
	本章介绍金融行业最常用的两个风险测度的计算/估算：风险价值(VaR)、信用价值调整(CVaR)。

### 10.4.1 风险价值

	风险价值(VaR)是最广泛使用的风险测度之一，也是饱受争议的测度之一。
	
	VaR是一个以货币单位(如美元、欧元、日元)表示的数字，表示在给定的时间周期中不超过某种置信度(概率)的损失(或者一个投资组合、股票头寸等)。
	
	VaR的优势在于其直观性，但是标准VaR测度在捕捉市场尾部风险方面有很大问题。
	
### 10.4.2 信用价值调整

	其他重要的风险测度是信用风险价值(CVaR)和从CVaR中派生而来的信用价值调整(CVA)。
	
	条件风险价值(CVaR:conditional value at risk)是一种较VaR更优的风险计量技术，其含义为在投资组合的损失超过某个给定VaR值的条件下，该投资组合的平均损失值。
