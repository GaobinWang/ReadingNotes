# 统计学

	我可以用统计学证明一切，除了真相。  --George Canning
	
	统计学是一个广泛的领域。该领域提供的工具和结果已经成为金融学中不可或缺的一部分。
	
	这也能够解释R等领域专用语言在金融行业中的流行。
	
	模型越精细、越复杂，易于使用和高性能的计算解决方案就越重要。
	
	本章聚焦于一下几个主题：正态性检验；投资组合理论；主成分分析；贝叶斯回归。
