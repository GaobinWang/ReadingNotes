## 2.2 Python工具

	编程语言的成功和流行，在某种程度上归功于改语言可用的工具。
	
	下面我们重点介绍当今最流行的两种工具:IPython和Spyder。

### 2.2.1 IPython

	利用IPython开发出的一个生态系统非常成功和具有吸引力，以至于其他语言的用户也利用它所提供的基本方法和构架。

	在金融环境中，IPython称为有价值的工具。
	
	例如，算法的数学描述和可执行Python版本可以存在于同一个文档。
	
	根据应用场景，可以生成一个网页(例如企业内联网)，PDF文件(例如客户邮件)或者演示(例如董事会)。

### 2.2.2 Spyder

	虽然IPython可以满足大部分用户的需求，但是较大的项目通常偶更高的要求。特别是IPython没有直接的内建编辑器。
	
	对于寻求更传统开发环境的人，Spyder可能是更好的选择。
