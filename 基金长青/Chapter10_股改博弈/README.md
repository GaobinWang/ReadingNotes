# 股改博弈

股权分置改革不仅是中国资本市场的一件大事，也是党中央、国务院的重大决策，开弓没有回头箭，必须搞好。  

---中国证监会前主席尚福林

2005年是中国股市否极泰来的一年。上半年，上证综指无情地击穿千点大关。经历四年多熊市煎熬，深度套牢的个人投资者彷徨，机构投资者无奈，中国股市路在何方？

山穷水复疑无路，柳暗花明又一村。股权分置改革的启动，划出中国股市新一轮牛熊转折的分水岭。

----

## 股权不改 股市不活

* 股权分置问题由来已久，是中国资本市场特定历史时期的产物。在计划经济体制下，为了尽快把资本市场建立起来，避免上市公司的国有资产流失，采取了“国有资产股份不上市流通，增量募集股份流通”的股权分置模式。

* 后来上市公司均沿袭了这一政策，股权被人为地割裂成非流通股和流通股，其中非流通股通常处于控股地位。

* 随着股市不断发展、上市公司越来越多，股权分置问题越来越严重。

* 2001-2005年长达4年多的熊市，是所有亲身经历的投资者至今无法忘怀的梦魇。造成中国股市长期低迷的原因很多，股权分置是一个重要因素。同股不同权，上市公司的治理结构存在严重缺陷，非流通股一股独大，上市公司缺乏诚信，控股股东利用控股权任意侵害中小股东利益的现象时有发生。

* 著名经济学家华生曾一针见血地指出：“中国股市最深层次的病根是圈钱，与国际接轨最大的障碍是股权分置。”


----

## 艰难的博弈

* 2005年5月10日，华生在《中国证券报》发表了《市场转折的信号--股权分置改革试点点评方案》一文，指出中国股票市场牛熊转换近在眼前，股权分置改革试点方案“方向正确，瑕不掩瑜”，现在“需要破釜沉舟，只能成功不能失败”。

* 第一批股改试点民营企业占了一半，且有一家国企没有成功。（三一重工、清华同方等）

* 第二批试点公司多为国有大型企业，在试点数量和公司代表性上都有了很大提高，坚定了社会各界对股改的信心。




----

## 基金实至名归

* 股权分置改革是中国资本市场完善市场基础制度和运行机制的一次根本性变革，被誉为中国股市的划时代里程碑。

* 关于股改的历史意义，尚福林主席在推动股改之处早有诠释--股权分置改革不是万能的，但不进行股权分置改革却是万万不能的。股权分置改革的意义在于不仅解决历史问题，还为资本市场各项改革和制度创新积累了经验、创造了条件。

* 股权分置改革只是一个良好的开端，并不能一劳永逸地解决公司治理的所有问题。中国社会仍缺乏尊重中小投资者的传统和股权文化。
