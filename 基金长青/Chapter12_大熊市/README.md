# 大熊市：泡沫破灭

2008年，A股在内忧外患中坠落。上涨综指大跌3450点，跌幅65.39%，史无前例。A股总市值缩水62.9%，投资者损失惨重。只有经历了2008年，基金经理才知道什么事大熊市，才能体会到大熊市对他们心理上的巨大冲击。

* “没有最低，只有更低”；“地板下面还有地下室，地下室下面还有地狱，地狱还有18层”成了自我解嘲的说辞。

* 尽管有“股市不言底”的股市谚语，亲眼所见、亲生经历时，还是令人目瞪口呆。


----

## 大崩溃：熊市不言底

### 反弹凋零 熊市降临

* 2008年1月2日--2008年1月14日，上证综指延续2007你那最后一周的上涨势头，从5265点上涨到5497点。

关键时刻，几个重大不利因素出现了：

（1）**宏观经济政策、流动性收紧。** 由于经济过热和通胀压力高企，2007年下半年央行连续多次提高存款基准利率与上调存款准备金率，持续收紧流动性，叠加效应逐渐显现。2008年初南方罕见的冰雪自然灾害造成的GDP损失也拉低了GDP增速。

（2）**美国次贷危机恶化。**

（3）**“大小非”解禁和再融资压力显现。** 2006-2007年是享受股改成果的幸福时光，2008年进入“后股改”成本支付阶段，全年“大小非”解禁市值将达3万亿元，占9万亿元流通市值的1/3。

基本面不利和市场供需失衡成为压倒疯牛的最后一根稻草。

* 2008年1月15日，A股市场掉头向下，熊市进场了。

祸不单行，外围市场大跌之时，A股市场上大量的“超级航母”相继传来巨额融资的消息。中国平安、浦发银行、中国人寿纷纷融资，加之金融股巨量“小非”大量套现，股市继续暴跌。

* 2008年2月1日，上证综指最低探至4195.75点，暴风骤雨式下跌才暂告一段落。仅14个交易日，上证综指暴跌24.03%，确认了6124.04点历史大顶，彻底终结了持续两年的超级大牛市行情，“上证综指上万点”的梦想彻底破灭。

虽然上证综指距顶点已经大跌2000点，但是很多投资者还没有意识到熊市已经来临，还幻想牛市继续。

* 2008年2月4日-2008年2月20日（2月6日-2月13日为春节），随着南方暴雪灾情好转和2日国际股市大幅反弹，在股票型基金审批开闸、融资融券业务适时试点等利好的推动下，沪深两市开始反弹。


* 2008年3月11日，国家统计局公布2月份的CPI同比上涨8.7%，创12年新高。

* 2008年3月18日，温家宝总理在十一届全国人大一次会议举行的记者招待会上表示：“今年我国经济可能是最困难的一年”，“在众多问题中排在第一位的是物价问题”。

* 2008年3月17日，有者85年历史的全球第五大投行贝尔斯登倒在次贷危机之下。


* 内交外困之下，两市一路震荡下跌至4月22日。从年初至4月22日，上证综指暴跌45.85%，深证成指暴跌44.92%。

面对股市持续暴跌，监管层坐立不安，开始干预股市。

* 2008年4月20日，中国证监会针对导致股市供求失衡的“大小非”减持出台规范措施。

* 2008年4月23日晚，财政部宣布从24日起印花税率由千分之三下调至千分之一。

* 2008年4月24日，两市双双大涨超过9%，涨停个股超过800多只，成交金额达到今年最大的1895.95亿元。这和2002年“6.24”行情非常相似的熊市从救市政策驱动的超跌反弹，只能影响短期走势，无法改变市场的中期趋势。

* 2008年5月6日，两市再次掉头向下。

* 2008年5月20日，国际油价飙升引发国际能源供应恐慌，沪深股市再次重挫，新一轮下跌开始。

* 2008年6月10日，市场受央行大幅上调存款准备金率1%，国际原油价格逼近140美元大关以及美股重挫400点等多重因素利空影响，股市大幅下跌。

* 2008年6月30日，上证综指报收2736.10点，上半年跌幅48%。

* 2008年7月，投资者希望借助奥运行情走出一波反弹，整个7月，市场平稳运行，上证综指小涨1.45%，投资者希望在接下来的奥运行情中有所收获。

* 2008年8月，奥运行情破灭，股市继续下跌。

“奥运”行情落空了。许多投资者认为，8月8日开始的大跌是预期奥运利好没有兑现所致。利好落空只是表面现象，股市下跌有更深、更复杂的原因。本轮下跌的根本原因在于经济中短期形式不明朗。虽然经济仍然保持快速增长，但上半年的CPI同比上涨7.9%，迫使政府加大宏观调控力度，加速投资者对经济增速放缓预期。同时，生产成本失控、国内外需求下降、国际能源价格保障等因素使本已严重的经济形势更加不利。复杂的国际政治局势增加了投资的不确定性，助长了奥运开幕后市场的悲观气氛。

* 2008年9月，国内外利空不断来袭。“大小非”加速减持、美国次贷危机持续恶化、美欧股市全面暴跌。内忧外患下股市缩量下跌。

* 2008年9月15日，雷曼兄弟申请破产保护、美林被收购、AIG信用评级下调，美国最大的住房抵押贷款银行之一华盛顿互惠银行也风雨飘摇。

* 2008年9月15日晚，中国人民银行紧急宣布下调一年期人民币贷款基准利率0.27个百分点，下调除五大行和邮政储蓄银行外的存款类金融机构存款准备金率0.5个百分点。9月16日，虽有货币政策放松利好刺激，也无法挽回股指坚决下行的心。

* 2008年9月16-18日，股市连续跌三天，个股惨不忍睹。政府相关部门坐不住了。

* 2008年18日晚间，国资委、财政部和中央汇金公司分别传来爆炸性消息。国资委网站发布信息，支持中央企业增持或回购上市公司股份；中央汇金投资公司宣布，将在二级市场自主购入工行、中行、建行的股票，并从即日起开始市场操作；同时，财政部、国家税务总局调整交易印花税为单边征收。

* 2008年9月19日，受三大政策利好刺激，沪深股指分别高开9.06%和8.35%。

* “9.19”“井喷”行情仅维持两天，市场开始大幅震荡。上证综指9月25日摸高2333.28点后再度掉头向下。

* 2008年10月28日，上证综指盘中跌至1664.93点，较历史高点6124.04点下跌了72.86%。


* 2008年11月5日，温家宝总理宣布，政府实施4万亿经济刺激计划。10日市场跳空高开，上证综指单日上涨7.27%，但屡战屡败的投资者再也不敢贸然抢反弹了。过度的恐惧使投资者没有意识到市场即将迎来转折。

2008年末，上证综指报收1820.81点，全年下跌65.39%。

## 冰火两重天 基金牛熊大反转

* 2007年下半年一时冲动买入基金的投资者2008年后悔不已。2007年那句“你不理财，财不理你”的口号传遍大江南北。

* 泡沫破灭，投资者财富缩水。尽管投资者对基金失望，却没有大规模赎回基金。这和中国个人投资者的行为习惯有关：当购买股票、基金被套后，往往不愿意立即卖出实现亏损，而是试图等待反弹解套。

## 顺势而为 保持轻仓

* 2008年2月的投委会月度会议上，我们对市场形势和投资策略进行了充分的、激烈的讨论。有的基金经理从房地产产业链的先导作用出发，前瞻性地判断由于物价、房地产调控形势严峻，房地产企业将收缩新开工，进而对水泥、钢铁、煤炭等下游产业产生负面影响，经济增速将出现显著回落，建议大力度减持股票的仓位。

* 8月11日基金经理晨会上，基金经理持续看空后市。王亚伟提出：“现在不应该再去分析基本面，更应看看市场情绪。”

## 熊市反思

----


## 基金经理熊市投资之道

幸运的是，亲身经历大熊市的惨痛失利后，基金经理收获了宝贵的教训，更深刻体会到宏观策略研判框架、风险控制的重要性，“自上而下”的资产配置意识、风险意识大为增强。经历一轮牛熊市的洗礼后，基金经理才能积攒足够的经验教训应对股市复杂的情况。

* **及时准确的预判宏观大势**
投资股票是投资未来。看清大方向、判断对大趋势是基金经理的基本功，是决定基金经理投资结果最重要的因素。基金经理如果仅擅长行业和个股分析、仅熟悉财务报表是远远不够的。基金经理在大方向判断上一旦发生大偏离，就会犯下战略性错误。2008年只会“自下而上”、不擅长“自上而下”，会大致致命的亏损。

* **针对性地配置资产**
对市场做出判断后，基金经理应心到手到，知行合一，采取与判断一致的资产配置策略。
* 在操作上，也不要幻想等待反弹再减仓，因为熊市第一波下跌是非常迅猛的，往往要到跌幅非常大了以后才会有反弹，及时顺势减仓的平均减持价格可能会高于反弹再减持的平均减持价格，减仓效果更好。

* 在风格方面，熊市应采取防御型的投资风格，如通过投资弱周期、低估值的股票增加组合的防御性，而强周期、高估值的股票往往跑输大市。

* 在行业配置方面，周期性行业，受行业景气度下滑和市场估值水平下降的双重打击，跌幅往往最惨，基金经理对这类股票应坚决回避；成长型企业，业绩持续稳定高增长的公司通常能够维持住估值水平，跌幅往往有限，在一些小规模的熊市中这类个股甚至能出现结构性上涨的局部牛市行情，对这类公司应更积极一些，尤其深入发掘增长速度被市场低估、存在预期差的个股；稳定类行业，如果业绩稳定（例如公共事业、交通运输）、估值水平较低、具有稳定分红，属于有安全边际的价值股，可以作为防御性品种加以配置、填充仓位替代现金以符合持股比例下限的要求。如果确认熊市结束，转入进攻，则不宜持有太多稳定类行业的股票。


* **独立思考，不随大流**
格雷厄姆说过，要想在证券市场上取得成功，第一要正确思考，第二要独立思考。

* **增强风险意识，严格止损操作**
在下跌成为随机事件的情况下，采取有效手段降低亏损概率是基金经理的重要工作。

* **保持良好的投资心态**
