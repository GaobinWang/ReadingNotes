# 如何成为优秀基金经理

追求业绩长青是每位基金经理的梦想。优秀的基金经理一定产生于优秀的基金管理公司。

* 优秀基金经理所具有的特质：理性，有责任心和耐心，专业素养深厚，形成一套系统的投资理念，能够真正挑选出具有长期投资价值的上市公司，能够对宏观经济大趋势做出前瞻性的判断，至少经过一轮以上牛熊更迭的洗礼，能够最大限度地跳出贪婪与恐惧的轮回。

## 业绩长青的投资大师


### 最成功的股票作手--杰西 利维摩尔

* 利弗莫尔投资职业生涯活跃期大致在1900-1930年，也是期间最有影响的投资大师之一。
* 从《股票大作手回忆录》中可以看出，利弗莫尔对人性和市场有着深刻的反省与透彻的把握，几十年来，一代又一代的投资者从这本书中获得教益。随着人生经历与投资经历的丰富，我们每隔一段时间阅读此书都会有新感悟与收获。




### 最伟大的投资大师--沃伦 巴菲特

* 巴菲特是历史上最伟大的投资大师。从1956年开始的半个世纪里，管理的资产增值了3万多倍。

* **独特的投资方法及人生经历：** 巴菲特继承并发展了格雷厄姆的价值投资理念和方法，融合了费雪的成长型投资理念，将价值投资的内涵从清算价值扩展到成长与行业壁垒。只做最优吸引力的交易是巴菲特投资方法的精髓，他始终能做到这一点与其独特的人生经历密不可分。

* 诚如巴菲特所言，“（深处华尔街的）缺点就是，在任何一个市场环境下，华尔街的情况都太极端了，你会被过度刺激，好像被逼着每天都要去做点什么。你应该做的是远离那些促使你仓促作出决定的环境。”

### 最优秀的基金经理--彼得 林奇

* 林奇是历史上最成功的基金经理。1977-1990年，林奇担任富达麦哲伦基金的基金经理。13年间，该基金称为最成功的基金，规模从2000万美元成长至140亿美元，是当时全球管理规模最大的基金。13年间基金净值年复合增长率达到29%，超过标普500指数15个百分点，对一名任期10年以上的基金经理来说，至今仍是最高记录。

* **独特的投资方法及人生经历：** 基金经理中，林奇“自上而下”投资的宽度与广度无人能及，这是麦哲伦基金达到巨大规模后仍能战胜市场的关键。麦哲伦基金后期达到100多亿美元，占美国股票市场1%。为了战胜市场，蓝筹股与热门行业中的热门股并非林奇重点投资的对象。后期麦哲伦基金投资的股票达到1400多只，包括很多不为其他基金关注的中小型股票，其中最大的100只股票占资产的一半，最大200只股票占资产2/3。
* 林奇的换手率比较高，平均一年至少要变化一次。林奇觉得他的工作越出色，换手率就越高。按彼得林奇自己的话说，他是个相对价值论者，而不是最大价值论者，“我从来没有做过任何大的决定，甚至从未想过要做出什么大的决定，我喜欢每天作出一些小决定。”
* 这种对海量信息迅速、准确的处理能力是绝大多数基金经理不具备的（可能这也是林奇隐退后麦哲伦基金渐趋平庸的原因之一），而这在相当程度上得益于林奇在富达研究主管位置上几年的历练：1974-1977年富达公司几级扩张行业研究领域，通过创建并管理大批新的行业研究领域，并于研究员深入讨论交流，林奇建立了自己对行业与公司进行简明分析的框架，并一次又一次地把研究的想法与实际投资结果相比较验证，从而逐步建立和培养了对公司价值及其变化进行迅速、简明、清晰判断的超凡能力。



### 最著名的对冲基金大鳄--乔治 索罗斯

* 索罗斯是著名的对冲基金经理，广为人知事迹分别是：1992年沽空英镑；1997年亚洲金融危机期间做空东南亚各国货币；2013年初做空日元。
* 不同于传统的纯多头基金经理，索罗斯的大部分财富通过做空获得。 他的投资哲学是寻找和发现宏观经济趋势的形成和转折，在市场拐点处，通过引入资金杠杆放大投资规模，引导甚至操纵市场方向创造巨额收益。


* **独特的投资方法及人生经历：** 索罗斯的“反身性”理论认为人们对事物的认识是不全面的，人们基于这种扭曲的认知而采取的行为可能会进一步强化原来的看法，形成自我强化趋势。

* “当有机会获利时，千万不要畏缩不前。当你对一笔交易有把握时，给对方致命一击，仅仅做对还不够，要尽可能多地获取。”

* 索罗斯对年轻时的磨难总结出两条生存技巧，也直接影响了他的金融生涯：第一是不要害怕冒险，第二是冒险时不要压上全部家当。

* **利弗莫尔**
利弗莫尔属于技术分析出身、遵循“高买高卖”、“低卖更低买”的趋势投资者，但他的趋势投资方法既非一成不变，也非一蹴而就。

* 在其投资生涯跃升的两个重要关口，都遭遇了挫折或失败。他从失败中领悟到现有方法的局限性，并迅速改善、丰富了自己的趋势投资方法，体现了伟大投资家的天才悟性与惊人的学习能力。
* 第一次失败他学习到两点：花很多时间研究市场趋势而不是市场的短期波动，包括看产业报告、财务与商业统计等基本面资料；耐心地坚守在主要趋势之中，不理会短期波动与反复，靠时间赚钱。

* 成名之后运作大资金集中于个股投资，从挫折中逐步领悟到大资金的投资成败根本上仍然取决于对整体市场的判断。利弗莫尔因此发展了一些自己判断市场的方法，尤其是紧密保持对市场龙头行业及龙头股的产业分析与股价走势分析，并将其作为判断大盘整体状态与走势的领先指标，进而指导个股投资。


* **巴菲特**
“耐心等待机会，只做最优吸引力的交易” 巴菲特的投资理念从未变过。但具体的投资方法经历了三个阶段的变化：选择有长期竞争壁垒的公司做积极股东--做最后的公司投资人--做最后的保险人。


* **彼得林奇**
* 林奇能够高效率、宽广地投资，一定程度上得益于各卖方纷纷在第一时间提供的上市公司的有价值信息：麦哲伦基金作为最大的、高换手率的基金，巨额交易佣金对华尔街券商的影响力不言而喻。麦哲伦巨大的基金规模也从劣势变成了有用信息获取上的优势。
* 林奇把公司分为6中类型：缓慢增长型、稳定增长型、快速增长型、周期型、困境反转型、隐蔽资产型。对这6类公司，林奇采用了不同的交易策略

* **缓慢增长型公司** 的典型代表是电力、公共事业公司，业绩增速不快、股票价格也不会很快上升，林奇会果断放弃。

* **稳定增长型公司** 如可口可乐、宝洁等，盈利年均增速约为10%-12%，在经济衰退或低迷时可以对组合起到较好的保护；若某只股票买入时机正确、买入价格合理、买入后涨幅较大，林奇会波段卖出，然后再选择尚未上涨的同类型股反复操作。
* **快速增长型公司** 是林奇最喜欢的股票类型之一，规模小、新成立不久、高成长、年均盈利增速约为20%-25%；只要能保持较快增速就会一直是股市中的赢家。
* **周期型公司** 被视为类似于“期权”的品种进行交易。
* **隐蔽资产型公司** 要善于发现，耐心等待。

* **索罗斯**
索罗斯上大学时深受导师卡尔 波普尔影响。科学方法的首要原理是完备知识的不可企及，也就是说，金融市场的有效性假说是不成立的。

* 索罗斯认为投资者对于事物的认识永远是片面的和不正确的，投资者的行为也不可能是完全正确的。市场不存在所谓的价格均衡，也不存在所谓的完全理性。投资者行为受其认识驱动，决定市场趋势的是投资者的主流偏向。很多时候主流倾向的形成会产生自我验证和自我加速现象，引导市场产生大的趋势运动。索罗斯追求的理想投资机会就是抓住这些正反馈和（或）负反馈机制的形成，利用投资者的偏见获得收益。
* 索罗斯是一个宏观驱动的投资者。在市场平稳或市场相对有效阶段，他不会建立大的风险头寸，而是静观其变，仿佛潜伏水底的鳄鱼。每当世界金融市场陷入混乱时，索罗斯的身影总会浮现，但他并不是一味蛮干。

### 平衡看待得失与平衡情绪的能力

* 对于投资失误，利弗莫尔懊恼的不是亏钱的结果，而是投资时没能抵抗住人性的诱惑，没有坚守投资纪律，偏离了自己的投资原则和方法。在他看来，在某项投资上亏钱是不可避免的事，亏再多的钱也不会让其懊恼，让他无法安眠的是自己的投资方法是不是出了问题。

* 对于巴菲特而言，投资跟打桥牌一样，都是简单、朴素、与压力无关、过程充满乐趣的事情，投资是生活中自然而然的一部分。林奇在事业与能力的高峰期选择急流勇退。由此也可见他们对待财富与名声都有一种健康、知止的基本价值观。

## 优秀基金经理的特质

投资是科学与艺术的结合。科学的部分相对容易理解，就是做好投资所需的基本技能。通过系统的学习和训练，这些技能是能够掌握的，但要做得比别人好也不容易。投资中的艺术部分就不太容易把握了。所谓投资中的艺术，是指投资中对人性的理解。人性不服从物理定律。人构成的市场中就不会有一成不变的事物。同一类型的信息可能导致不同的结果，比如一家公司业绩亏损可能会导致股价大跌，但有时候业绩亏损则被认为是利空出尽，反而成为股价上涨的契机。这种相互矛盾的现象在市场中屡见不鲜。如何正确的理解和判断这些因素，就是投资的艺术。

### 始终对市场充满敬畏之心

* 投资的本质是要回答并解决“人与市场”的关系。
* 基金经理要对市场充满敬畏之心，善居其下；同时也要找到市场自身规律，在掌握市场规律的基础上探索与市场和谐相处之道。

### 始终保持自知之明，了解自己的能力边界

* 基金经理应该清楚自己的“能力圈”，明白自己擅长什么，不擅长什么；在什么类型的股票上认识超越常人而能够赚钱，在什么类型的股票上认识不足而不该投资。基金经理经过一段时间的投资实践后，应该注重功过成败的总结。了解自己的特色，划出自己的“能力圈”，形成自己的投资风格。

### 独立思考 把握好变与不变的分寸

* 投资本身是一个充满矛盾的过程。优秀的基金经理需要处理好坚守与放弃、深度与广度、共性与个性、开放与保守、灵敏与拙朴等几对矛盾关系。真理多走一步就会变成谬误，如何把握好这些矛盾关系的度与分寸既有艺术的成分，也与每个人的气质与经历有关。
* 要想长期持续的战胜市场，基金经理必然要有一些本质的共性的东西；但市场的环境与结构不断在变，想要偏离而不脱离市场，也需要不断变化。难就难在度的把握。
* 每个曾经成功过的基金经理都会试着去摸索适合自己的变与不变的方式与分寸，给出自己的答案，这也是基金经理从平凡走向卓越的必经阶段。

###  不断学习 与时俱进

* 与寻找新的投资品种相比，在市场系统结构、组合构建方法、投资策略、新的投资工具和方法等方面积极思考，寻求变化的机会，更有可能获得局部的创造性突破。
* 一名优秀的基金经理必然是能顺应时代趋势，把握转型方向的基金经理。

### 持之以恒地提高个人修养，坚持不懈地自觉修行

* 基金经理的眼光与修养决定了未来的高度。

* 知易行难。哪怕明白所有道理也不等于有智慧。平常所谓的“聪明”，只是悟性好。如果在重要关头不能把持住自己，这种聪明也仅是小聪明。只有能身体力行，重要关头能把持住才是“智慧”。
* 中国佛教两个流派--禅宗与天台宗对智慧的态度不同。禅宗强调“顿悟”，天台宗强调修行。
* 对以业绩长青为目的基金经理，靠“顿悟”得来的智慧与道理是不可靠的，只有长期坚持不懈的严格修行与自省出来的道理才能最终成为自觉的行为。儒家“吾日三省吾身”应该就是这个道理。

* 基金经理时常需要跟人性作斗争。修行的价值何在？跟滑雪一样，当你一次次克服内心的恐惧作出正确的反应动作，你会慢慢体会到正确的动作带给你的快感与乐趣，从而把正确的动作最终沉淀为自觉的行为。

## 如何选拔优秀的基金经理

选拔优秀的基金经理需要良好的判断力和科学的选拔方法。以下几点在选拔基金经理方面尤为重要：

* **基金经理素质**
真正优秀的基金经理并不需要顶级智商，而是需要较好的天赋。后者是有非常复杂的因素决定的。

* **投资理念**
优秀的基金经理往往有鲜明的特征，其理念往往具有独特性，与众不同。

* **资产组合**
资产组合的变动严格遵循事前制定的投资流程，体现了基金经理不需要依靠临时操作的自信，这样的基金经理将更值得信赖。
