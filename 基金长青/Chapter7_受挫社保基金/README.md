# 受挫社保基金

社会保障基金对资产的安全性、流动性及收益性有很高的要求。华夏基金在社保基金的投资管理中，出师不利，摔了跟头。

----

## 中石化配售

2000年中国政府成立全国社会保障基金，是朱F基总理任期内做的一件关系国计民生、影响深远的大事，开启了中国养老金投资管理的新时代。

社保基金如何战胜通货膨胀，实现保值增值是一件重要的事情。从发达国家养老金管理经验啦看，养老资金想长期获得超越通货膨胀的收益率，进入股票市场是必然的选择。

从基金公司业务发展的角度，从西方发达国家养老金管理，特别是美国401（k）计划的发展经验来看，养老金能够为基金公司提供长期稳定的资金来源，基金公司在养老金管理领域大有可为。社保基金的设立正是中国建立现代化社会保障体系以及探索养老金管理的起点，对中国基金公司及股票市场的发展具有重要意义。

----
## 社保基金首次“选秀”

社保基金首次“选秀”为基金公司多元化的资产管理开辟了道路，基金公司的业务范围从此就不再仅限于公募基金管理了。而且，基金公司成为社保基金管理人，有利于增强基金公司的竞争力，提升基金公司的品牌形象，有利于基金公司做大做强。

----

## 重建投研团队

投资管理能力是基金公司的核心竞争力，对投资研究投入多少都不为过。

社保基金的建立和发展，不仅为基金带来了上千亿元的资金，更重要的是，社保基金高标准、严要求，促进了基金经理理性投资理念的形成，提升了基金公司的投资管理水平。
